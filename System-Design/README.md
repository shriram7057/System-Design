# FAANG-Level System Design in Java

This repository demonstrates **production-grade System Design** using Java.
It covers Low-Level Design (LLD), concurrency, distributed systems, and scalability concepts.

## Coverage
- OOP & SOLID
- Design Patterns
- Parking Lot & Elevator LLD
- Thread Safety & Concurrency
- Load Balancer & Rate Limiter
- UML Diagrams

Built for **FAANG interviews and real-world systems**.
